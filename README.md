# Multiple Disease Prediction System

## 📌 Overview
This project is a web-based application that predicts multiple diseases using Machine Learning models. Users can input medical parameters and get instant predictions.

---

## 🚀 Features
- Predicts multiple diseases (Heart, Diabetes, etc.)
- User-friendly interface
- Real-time predictions
- Machine Learning models integrated with web application

---

## 🛠 Tech Stack
- Frontend: HTML, CSS
- Backend: Python, Django
- Machine Learning: Scikit-learn
- Database: SQLite

---

## 📂 Project Structure
- UIdesign/ → Frontend UI files  
- images/ → Screenshots of project  
- models/ → ML models (if added)  
- templates/ → HTML files  

---

## ⚙️ Installation

### 1. Clone the repository
git clone https://github.com/sinchanats29/multiple-disease-prediction.git

2. Navigate to project folder
cd multiple-disease-prediction

4. Install dependencies
pip install -r requirements.txt
Run the project
python manage.py runserver

This project helps in early disease prediction and demonstrates the use of Machine Learning in healthcare applications.
